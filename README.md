# simple chat js

[![travis status](https://travis-ci.org/sophietk/test-ci.svg?branch=heroku)](https://travis-ci.org/sophietk/test-ci)

technos:

* tools: node, grunt, bower, sass
* libs: backbone, zepto, lodash, materialize

## install

```
gem install sass
npm install -g grunt-cli
npm install -g bower
npm install
bower install
```

## start dev server

```
npm start
```

## build prod files

```
grunt build
grunt dist
```
